## Product Goal

让课程站成为“线上稳定页可保留、课程新增页可持续增量发布”的可靠发布面，同时把视觉切到白色模式，提升白天阅读和截图分享体验。

## Users

- 站点维护者：需要安全地把新课程笔记上站，不想误覆盖旧内容。
- 课程笔记读者：需要继续访问旧页，并看到新增课程页与更清爽的白色主题。

## User Stories

- 作为维护者，我希望站点能先吃线上稳定基线，再增量并入本地新页，这样旧内容不会被重新生成坏。
- 作为维护者，我希望新课次一旦本地确认“有图已跑好”，就能稳定发布到站点。
- 作为读者，我希望站点改成白色模式后，仍然保留课程站的结构感和阅读导向，而不是变成普通亮色模板。

## Non-Goals

- 不重新跑课程 OCR / MinerU。
- 不改课程正文内容。
- 不重做整站 IA 或换框架。

## Functional Requirements

- 从线上课程站抓取稳定旧页，重建可供本地生成器消费的 staging source。
- 从本地课程目录识别已完成且线上缺失的课次，并转换成 staging source 增量输入。
- 构建产物继续输出课程 HTML、Markdown、sitemap 与 course-assets。
- 支持新科目 `安全学原理`。

## Visual Requirements

- 提供白色模式。
- 背景保留轻量 grid / dot-grid 纹理。
- 维持课程站的档案式阅读秩序，不做花哨营销风。

## Acceptance

- 本地构建后，抽样旧页内容不变形。
- 新增完成页在首页与详情页都可见。
- 白色模式在首页、课程索引页、详情页、Markdown 关联样式中一致生效。
